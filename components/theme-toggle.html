ThemeToggle

Cycles through light → dark → auto color scheme.

Description

Cycles through light → dark → auto color scheme. Replaces ThemeSelector from the source. Renders as an action icon button showing the current theme state.

Where this component exists in the Prometheus UI
View in Prometheus UI → /

ThemeToggle in the app header — light/dark mode switch

Accessibility

  • Button has descriptive aria-label showing current and next state
  • Cycles through three states: light → dark → auto → light

Do / Don’t

✓ Do
  • Application header — theme switching
  • Any context where users need to control color scheme
  • Place ThemeToggle in the header actions area. Let it manage color scheme state internally.
✗ Don’t
  • Page-level theme sections
  • Content-specific color controls
  • Don't create separate theme switching mechanisms outside of ThemeToggle.
