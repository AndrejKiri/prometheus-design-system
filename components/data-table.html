DataTable

Generic sortable data display table.

Description

Generic sortable data display table. Supports custom cell rendering, column sorting, empty states, and horizontal scroll. Used throughout Prometheus UI for flags, statistics, and tabular data display.

Where this component exists in the Prometheus UI
View in Prometheus UI → /flags

DataTable on the Flags page — sortable table with flag values

View in Prometheus UI → /tsdb-status

DataTable on the TSDB Status page — top 10 metrics

Code Example

import { DataTable } from "@prometheus-io/design-system";

<DataTable
  columns={[
    { key: "flag", header: "Flag", sortable: true },
    { key: "value", header: "Value" },
  ]}
  data={flags}
  rowKey="flag"
  emptyMessage="No flags found"
  sortKey={sortKey}
  sortDirection={sortDirection}
  onSort={handleSort}
/>

Accessibility

  • Sortable column headers have role="button" and tabIndex={0}
  • Sort state communicated via aria-sort attribute
  • Keyboard support: Enter/Space to toggle sort on column headers
  • Empty state uses centered dimmed text

Do / Don’t

✓ Do
  • Displaying tabular data (flags, stats, TSDB info)
  • Sortable columns with visual sort indicators
  • Tables with custom cell renderers (badges, links, formatted values)
  • Use DataTable for all tabular data. Use column render for custom cell content. Use rowKey for stable React keys.
✗ Don’t
  • Key-value pair display (use KeyValueTable)
  • Complex interactive grids with editing capabilities
  • Don't build custom <Table> layouts with manual sort logic. Don't use array index as key without rowKey.
