Framework Notes

UI is built with Mantine v7+ (html[data-mantine-color-scheme] light|dark). Syntax highlighting uses highlight.js (hljs-* classes). Primary brand color is Mantine blue rgb(34,139,230) in light / rgb(25,113,194) in dark. Nav bar is a fixed dark slate rgb(65,73,81) that is identical across both themes.

Screenshot Limitation

Screenshots in screenshots/ are 1x1 placeholder JPEGs (196 bytes each). The Cowork sandbox's javascript_tool blocks base64/hex image data returning from html2canvas, so rendered page imagery could not be exfiltrated through the browser automation channel. All observations in this report are derived from live DOM inspection via getComputedStyle, class-name surveys, and in-context screenshot viewing through the Chrome MCP extension — not from post-hoc image analysis. A downstream consumer that needs real pixel-level screenshots should re-capture them using a local Playwright/Puppeteer script against the same URL.
