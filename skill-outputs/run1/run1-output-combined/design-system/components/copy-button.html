Copy Button

Copy-to-clipboard action icon

✓ DoShow a brief 'copied' visual feedback state (checkmark icon, green color) for 2s after copy.
✗ Don'tDon't show a toast/notification for a copy action — the in-button feedback is sufficient.
✓ DoUse Mantine CopyButton wrapper to handle clipboard API and feedback state.
✗ Don'tDon't place the copy button inside the scrollable pre block — position it above the content for constant visibility.

Description

Mantine ActionIcon with a clipboard icon, positioned top-right of the config card on /config. Part of the ActionIcon primitive family also used in the navbar (theme toggle, settings, docs).

simple input /config

Design variants

default

Clipboard icon ActionIcon, idle state.

Tokens: brand-blue-light

copied

Checkmark feedback state after successful copy.

Tokens: ok

Implementation

<CopyButton value={configYaml} timeout={2000}>
  {({ copied, copy }) => (
    <ActionIcon
      aria-label={copied ? 'Copied!' : 'Copy configuration'}
      color={copied ? 'green' : 'blue'}
      variant="subtle"
      onClick={copy}
    >
      {copied ? <IconCheck size={18} /> : <IconClipboard size={18} />}
    </ActionIcon>
  )}
</CopyButton>

Accessibility

ARIAUse aria-label='Copy configuration' on the ActionIcon.
KeyboardFocusable via Tab, activatable via Enter/Space.
Screen readerUpdate aria-label to 'Copied!' during feedback state so screen readers announce the result.
