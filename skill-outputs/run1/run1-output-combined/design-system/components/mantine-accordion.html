Mantine Accordion

Collapsible scrape-pool group

✓ DoUse accordion to group logically related rows that users may want to collapse independently.
✗ Don'tDon't use accordion for fewer than 3 groups — flat list is simpler to scan.
✓ DoPut summary badges in the accordion header so users can scan without expanding.

Description

Collapsible sections used on /targets to group scrape pools. Each accordion header shows the pool name plus Stats Badges summarizing endpoint counts. Body reveals the target table for that pool.
