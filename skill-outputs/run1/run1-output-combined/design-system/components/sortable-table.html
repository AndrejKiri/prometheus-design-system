Sortable Table

Column-sortable data table

✓ DoShow the stacked chevron icon on every sortable column header so users know they can click.
✗ Don'tDon't make tables sortable only in some pages — apply sort consistently where it adds value (INC-005).
✓ DoHighlight the currently active sort column header.

Description

Mantine table with clickable column headers for sorting. Uses a stacked chevron-up + chevron-down icon pair to signal sortability. Only /flags uses this — other tables are static (see INC-005).
