Key-Value Table

Property definition table

✓ DoUse this pattern for static metadata — build info, config values, system properties.
✗ Don'tDon't use this for large data sets — use Sortable Table instead.
✓ DoShow status values like 'Successful' as colored pills rather than plain text (INC-006).
✗ Don'tDon't render status text as plain prose when a health badge conveys the same information more clearly (INC-006).

Description

2-column definition-list table with bold key column and regular value column, 1px row dividers. Used for build info, runtime info, and TSDB head status. Static — no sort controls.
