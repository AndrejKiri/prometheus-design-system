Discovered/Target Labels Layout

Side-by-side label comparison layout

✓ DoAlways show both columns even if one is empty — the comparison is the UX.
✗ Don'tDon't collapse to a single column on non-mobile viewports — the side-by-side comparison is the point.
✓ DoUse the same Label Badge component for both columns.

Description

Two-column side-by-side layout on /service-discovery. Left column shows 'Discovered Labels' and right column shows 'Target Labels' for each discovered endpoint, both rendered as groups of Label Badges.
