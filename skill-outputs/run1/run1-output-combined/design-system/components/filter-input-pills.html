Filter Input (Pills)

Multi-chip filter toolbar

✓ DoUse this for list pages where users need to filter by multiple label dimensions simultaneously.
✗ Don'tDon't use this where a simple text match is sufficient — consider the single-field filter instead (INC-004).
✓ DoKeep placeholder text descriptive of what can be filtered (e.g. 'Filter by rule name or labels').
✗ Don'tDon't use the same placeholder on different pages if the filterable fields differ.

Description

Multi-filter toolbar built with Mantine PillsInput that lets users add multiple chip-style labeled filters. Used across four pages for label-oriented filtering. Placeholder text is shared verbatim between /alerts and /rules.
