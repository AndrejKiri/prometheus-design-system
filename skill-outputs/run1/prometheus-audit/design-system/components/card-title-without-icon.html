Card Title without Icon

Plain text card header

✓ DoUse plain titles for cards where no icon adds semantic value.
✗ Don'tDon't mix icon and plain card titles on the same page — pick one style and apply it consistently (see INC-001).

Description

Card header variant that renders only the title text with no leading icon. Used on /tsdb-status and /flags. Visually lighter than the icon variant and creates an inconsistency with icon-title cards elsewhere.
