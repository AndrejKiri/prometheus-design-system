Header/Navbar

Global app shell navigation

✓ DoKeep the navbar background rgb(65,73,81) identical in light and dark themes — it is a product-brand anchor.
✗ Don'tDon't change the navbar background based on color scheme — it breaks the brand identity.
✓ DoUse Tabler Icons for all ActionIcons to stay consistent with the rest of the icon set.
✗ Don'tDon't add more than 4 primary route links without a dropdown or secondary nav.

Description

Fixed top navigation bar present on every page. Contains the Prometheus logo, primary route links, and right-side utility ActionIcons. Background is a dark slate that does not change between light and dark themes.

Accessibility

ARIATheme toggle ActionIcon uses aria-label that flips between 'Switch to light theme' / 'Switch to dark theme'.
KeyboardTab through nav links and ActionIcons. Active route link should have aria-current='page'.
ContrastWhite text on rgb(65,73,81) slate passes WCAG AA (contrast ~5.1:1).
