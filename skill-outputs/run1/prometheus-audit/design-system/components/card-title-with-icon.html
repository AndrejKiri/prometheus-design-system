Card Title with Icon

Titled card header with leading icon

✓ DoChoose an icon that semantically relates to the card content (database → build, bell → alerts).
✗ Don'tDon't use decorative or random icons — every icon should communicate something about the card's subject.
✓ DoApply this pattern consistently to all cards that currently have it — and consider extending it to cards without icons (INC-001).
✗ Don'tDon't mix icon-title cards and plain-title cards on the same page without visual justification.

Description

Card header variant that places a semantically relevant Tabler icon immediately before the title text. Used on /status (Build, Runtime) and /alertmanager-discovery (Active, Dropped).
