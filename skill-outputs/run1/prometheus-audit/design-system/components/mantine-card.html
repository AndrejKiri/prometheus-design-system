Mantine Card

Content section container

✓ DoUse Mantine Card as the single wrapping container for any page-level content section.
✗ Don'tDon't nest cards inside cards — use Accordion or Table rows for sub-grouping instead.
✓ DoMaintain 16px internal padding consistently — do not override with tighter values.
✗ Don'tDon't use the card border-radius token (4px) for pill/badge shapes — use the pill token (1000px).

Description

Primary content container used on 7 of 10 pages. Wraps all major content sections with a consistent white/dark background, 1px border, 4px corner radius, and a subtle box shadow.
