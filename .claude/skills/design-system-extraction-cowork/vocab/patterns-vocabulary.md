## Naming rules (summary)

- Title Case noun-phrase: `Primary Button` not `primary button` or `Buttons (Primary)`
- Parenthetical variants: `Filter Input (Pills)` vs `Filter Input (Single Field)`
- Framework-prefix only when it disambiguates: `Mantine Accordion`, `Mantine Card`
- Lock names after the first page — reuse exactly
